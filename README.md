# Patient PATH

Patient PATH (Patient Transport Hub) is a comprehensive platform designed to streamline the coordination between hospitals and ambulance services for patient transport.

## Features

- **Hospital Dashboard**: Manage transport requests, track active transports, and view transport history
- **Ambulance Dashboard**: Manage fleet, respond to transport requests, and track assignments
- **Real-time Bidding System**: Ambulance services can bid on transport requests
- **GPS Tracking**: Real-time location tracking for ambulances
- **Transport History**: Comprehensive history of all transports for both hospitals and ambulance services

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase (Authentication & Database)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/patient-path.git
cd patient-path
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
